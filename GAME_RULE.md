# 게임 규칙: 과일 받아먹기 (Fruit Catcher)

## 🎮 게임 제목
**Fruit Catcher (과일 받아먹기)**

## 📝 게임 설명
하늘에서 떨어지는 맛있는 과일들을 창으로 받아내는 게임입니다!
몸을 왼쪽, 오른쪽으로 기울여 창을 이동시키고, 폭탄은 피해야 합니다.

## 🎯 게임 목표
- **시간 제한 없음!** 무제한으로 즐기세요.
- 최대한 오래 살아남아 점수를 쌓으세요.
- 폭탄을 받으면 점수가 깎이거나 생명이 줄어듭니다.
- 목표 점수: 500점 이상 달성!

---

## 🕹️ 게임 규칙

### 기본 조작 (Teachable Machine 포즈)
플레이어는 **바구니**를 조종합니다.

| 포즈 | 동작 | 설명 |
|---|---|---|
| **왼쪽 (Left)** | 바구니 왼쪽 이동 | 몸을 왼쪽으로 기울이거나 왼손을 듭니다. |
| **오른쪽 (Right)** | 바구니 오른쪽 이동 | 몸을 오른쪽으로 기울이거나 오른손을 듭니다. |
| **중앙 (Center)** | 정지 / 대기 | 정면을 보고 가만히 있습니다. |

### 아이템 및 점수

| 아이템 | 이미지(예시) | 점수 | 효과 |
|---|---|---|---|
| **사과 (Apple)** | 🍎 | +100점 | 기본 점수 획득 |
| **바나나 (Banana)** | 🍌 | +200점 | 더 높은 점수, 속도가 조금 빠름 |
| **포도 (Grape)** | 🍇 | +300점 | **고득점**, 매우 빠른 속도! |
| **이벤트 박스 (Event Box)** | 🎁 | - | **30초간 무적** (폭탄 방어) |
| **권총 (Gun)** | 🔫 | - | **15초간 슈팅 모드** (총알 발사) |
| **치킨 (Chicken)** | 🍗 | +1000점 | **대박 점수!** (5% 확률) |
| **폭탄 (Bomb)** | 💣 | -250점 | 점수 차감 및 화면 흔들림 효과 |

### 게임 진행
1. 게임이 시작되면 과일과 폭탄이 무작위 위치에서 떨어집니다.
2. 플레이어는 포즈를 취해 바구니를 좌우로 움직입니다.
3. 바구니에 과일이 닿으면 점수 획득! (+효과음)
4. 폭탄에 닿으면 점수 차감! (+꽝 소리)
5. **Endless Mode**: 시간 제한 없이 계속 진행됩니다. 지칠 때까지(또는 Stop 버튼을 누를 때까지) 해보세요!

### 레벨 시스템
- **Lv.1 (0~499점)**: 과일이 천천히 떨어짐, 폭탄 없음.
- **Lv.2 (500~999점)**: 과일 속도 증가, 폭탄 등장!
- **Lv.3 (1000점~)**: 과일과 폭탄이 비 오듯이 쏟아짐!

## 🎨 필요한 포즈 목록 (Teachable Machine 학습용)

Teachable Machine에서 아래 3가지 클래스(Class)를 학습시켜주세요.

1. **Left** (왼쪽으로 이동)
2. **Right** (오른쪽으로 이동)
3. **Center** (가만히 있기)

> **팁**: '왼쪽', '오른쪽'이라는 한글 이름 대신 영문 **Left, Right, Center**로 클래스 이름을 정하면 코드를 수정할 필요 없이 바로 인식됩니다!
