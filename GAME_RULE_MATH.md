# 지루한 수학 퀴즈 (Boring Math Quiz)

## ✏️ 게임 소개
잠들지 마세요! 1분 동안 당신의 수학 실력을 테스트하는 간단하지만 강력한 퀴즈 게임입니다.
초등, 중등, 고등 3가지 난이도에 도전하여 최고 점수를 획득하세요!

## 🎮 조작 방법
- **키보드 숫자패드**: 정답 입력
- **Backspace**: 입력 지우기
- **Enter**: 정답 제출
- **모른다 버튼 (또는 클릭)**: 문제 패스 (-2점)

## 📋 규칙
1. **제한 시간 60초** 동안 최대한 많은 문제를 풀어야 합니다.
2. 정답을 맞추면 **+10점**을 획득합니다.
3. 오답을 입력하면 **-5점** 감점됩니다.
4. 문제를 패스하면 **-2점** 감점됩니다.
5. 정답 확인 및 해설이 제공되면 잠시 후 다음 문제로 넘어갑니다.
6. 시간이 0이 되면 게임이 종료됩니다.

## 🎓 난이도 (Level)

### 1단계: 초등 (Elementary) - "기초 다지기" 🐥
- **대상**: 초등학교 1~6학년 수준
- **내용**: 
    - 덧셈, 뺄셈, 곱셈 (구구단)
    - 나눗셈 (나머지 없는 경우)

### 2단계: 중등 (Middle) - "공식의 시작" 🧑‍🏫
- **대상**: 중학교 1~3학년 수준
- **내용**: 
    - 일차방정식 (ax + b = c)
    - 거듭제곱 (a^b)
    - 음수 연산

### 3단계: 고등 (High) - "수학의 정석" 🎓
- **대상**: 고등학교 1~3학년 수준
- **내용**: 
    - 로그 (log)
    - 미분 (다항함수의 미분 계수)
    - 등차수열
