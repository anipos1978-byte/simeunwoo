# 게임 규칙: 버드스트라이크 피하기 (Bird Strike Dodge)

## 🎮 게임 제목
**버드스트라이크 피하기 ✈️**

## 📝 게임 설명
비행기를 조종하여 하늘에서 날아오는 새들을 피하는 게임입니다!
키보드 방향키로 비행기를 3개 구역(레인)에서 이동시키고, 별과 다이아를 모으세요.

## 🎯 게임 목표
- **시간 제한 없음!** 무제한으로 즐기세요.
- 새(🐦, 🦅, 🦆)를 피해 최대한 높은 점수를 노리세요!
- 별(⭐)과 다이아(💎), 치킨(🍗)을 모아 점수를 올리세요.
- 새에 부딪히면 점수가 -200점 깎입니다!

---

## 🕹️ 게임 규칙

### 기본 조작 (키보드)
비행기는 **3개 레인**(왼쪽/중앙/오른쪽)을 이동합니다.

| 키 | 동작 | 설명 |
|---|---|---|
| **← (왼쪽 방향키)** | 왼쪽 레인으로 이동 | A 키도 사용 가능 |
| **→ (오른쪽 방향키)** | 오른쪽 레인으로 이동 | D 키도 사용 가능 |

### 아이템 및 점수

| 아이템 | 이미지 | 점수/효과 | 등장 확률 |
|---|---|---|---|
| **새 (Bird)** | 🐦🦆 | -200점 | 55% |
| **큰 새 (Big Bird)** | 🦅 | -200점 (크기 큼) | 10% |
| **별 (Star)** | ⭐ | +100점 | 15% |
| **다이아 (Diamond)** | 💎 | +300점 | 10% |
| **치킨 (Chicken)** | 🍗 | +1000점 | 10% |
| **새 피함 보너스** | - | +10점 | 자동 |

### 게임 진행
1. 게임이 시작되면 비행기가 중앙 레인에서 시작합니다.
2. 키보드 방향키(← →)로 3개 레인을 이동합니다.
3. 새들은 3개 레인에서 직선으로 떨어집니다.
4. 새에 부딪히면 -200점! (1.5초간 무적)
5. 별/다이아/치킨을 먹으면 점수 획득!
6. **Endless Mode**: 게임 오버 없이 무한으로 즐기세요!

### 레벨 시스템
- **Lv.1 (0~499점)**: 새가 느리게 떨어짐
- **Lv.2 (500~999점)**: 새 속도 증가, 생성 빈도 증가
- **Lv.3 (1000점~)**: 새가 빠르게 쏟아짐!

---

## ⌨️ 조작 안내

> **팁**: 방향키 대신 **A키**(왼쪽), **D키**(오른쪽)로도 조작할 수 있습니다!
