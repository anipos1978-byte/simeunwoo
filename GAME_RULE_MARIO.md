# 🍄 마리오 탈출 (Mario Escape) 게임 규칙

정사각형 공터에서 버섯(굼바)들을 피해 최대한 오래 살아남으며 점수를 획득하는 서바이벌 게임입니다.

## 🕹️ 조작 방법
- **상하좌우 방향키 (↑↓←→)**: 마리오 이동
- **스페이스 바 (Space)**: 총 발사 (총 아이템 획득 시)

## 📝 점수 규칙
- **생존 점수**: 살아남아 있는 동안 점수가 계속 소량 증가합니다.
- **🪙 코인**: 획득 시 **+100점**
- **⭐ 슈퍼 스타**: 획득 시 **+500점** + 주변의 적들을 **모두 제거**!
- **🔫 총**: 획득 시 **총알 10발** 충전 (스페이스 바로 사격 가능)
- **💥 적 처치**: 총으로 버섯을 맞추면 **+50점**
- **🍄 버섯(Goomba) 충돌**: 게임 오버! (정지 후 다시 시작)

## 🆙 난이도 상승
- 점수가 **1000점** 단위로 오를 때마다 레벨이 상승합니다.
- 레벨이 오를수록 새로운 버섯이 더 빠르게 나타나며 이동 속도도 빨라집니다.
- 버섯은 벽에 부딪히면 튕겨 나오니 조심하세요!

## 💡 팁
- 슈퍼 스타는 위기의 순간을 넘기는 데 아주 유용합니다.
- 적들이 많아지면 중앙보다는 구석을 잘 활용해 보세요 (하지만 갇히지 않게 조심!)
