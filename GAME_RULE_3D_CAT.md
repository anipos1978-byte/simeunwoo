# 게임 규칙: 우당탕탕 고양이 3D (Naughty Cat 3D)

## 🐈 게임 소개
당신은 평범한 가정집의 사고뭉치 고양이입니다! 
할머니가 낮잠을 주무시거나 다른 방에 계시는 동안, 최대한 많은 물건을 떨어뜨리고 깨뜨려야 합니다.
하지만 조심하세요! 할머니에게 들키면 게임 오버입니다.

## 🎯 게임 목표
- **제한 시간 내 최고 점수 달성**: 고양이가 물건을 깨뜨릴 때마다 점수가 올라갑니다.
- **할머니 피하기**: 할머니의 시야에 들어오거나, 큰 소리를 내어 할머니를 깨우면 안 됩니다.
- **최종 목표**: 집안을 엉망진창으로 만들고 무사히 탈출하세요!

## 🕹️ 조작 방법 (3D 조작)
본 게임은 **Keyboard** 또는 **Pose 인식**으로 조작할 수 있습니다.

### [Keyboard]
- **W, A, S, D** 또는 **방향키**: 고양이 이동
- **Space**: 점프 (높은 곳에 올라가기)
- **E / Enter**: 물건 밀치기 (스포트라이트가 있을 때)

### [Pose Recognition (Teachable Machine)]
| 포즈 | 동작 |
|---|---|
| **Center** | 정지 / 앞으로 걷기 |
| **Left / Right** | 좌우 회전 |
| **Jump (양손 들기)** | 점프 |

## 📋 규칙 및 시스템

### 1. 아이템 및 점수
| 아이템 | 점수 | 설명 |
|---|---|---|
| **꽃병 (Vase)** | 100점 | 잘 깨지지만 소리가 큼 |
| **접시 (Plate)** | 50점 | 주방에 무더기로 있음 |
| **TV / 모니터** | 500점 | 아주 비싸고 큰 소리가 남 |
| **두루마리 휴지** | 30점 | 풀리면서 흔적을 남김 |

### 2. 할머니 AI (Grandma Presence)
- **청각 게이지 (Noise Meter)**: 물건이 깨질 때마다 게이지가 올라갑니다. 게이지가 꽉 차면 할머니가 방으로 달려옵니다!
- **시야 (Line of Sight)**: 할머니가 순찰하는 동안 붉은색 시야 범위에 들어가면 즉시 추격을 시작합니다.
- **숨기 (Hiding)**: 침대 밑이나 박스 안에 들어가면 할머니의 시야를 피할 수 있습니다.

### 3. 게임 종료 조건
1. 할머니에게 붙잡혔을 때.
2. 제한 시간이 모두 종료되었을 때.
